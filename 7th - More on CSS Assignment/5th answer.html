Google Fonts

Google Fonts is a service that offers free and open source font families, an interactive directory, and APIs for using the fonts with CSS and Android. It was previously known as Google Web Fonts.

Some of the best Google Fonts include: Roboto, Open Sans, Lato, Montserrat, Oswald.

Other fonts available on Google Fonts include: DM Sans, Inter, Space Mono, Space Grotesk, Work Sans, Syne, Libre Franklin, Cormorant.
